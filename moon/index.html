Your Random Code Pick: Binary Varshamov-Tenengolts (VT) code

Go to code → Refresh

Nearly optimal binary deletion-correcting code. Given integers \(n\geq 1\) and \(a\in\{0,1,\dots,n\}\), the associated binary Varshamov-Tenengolts code \(C_{n,a}\) corresponds to the set $$C_{n,a} = \left\{ x\in\{0,1\}^n : \sum_{i=1}^n ix_i = a \mod(n+1)\right\}.$$
